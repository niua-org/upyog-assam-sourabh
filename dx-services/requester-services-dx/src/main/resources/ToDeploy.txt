
First mvc clean:
mvn clean package -DskipTests  

Docker build:
docker build -t abhijdock123/requester-services-dx:uat-13-11-25-5 -f Dockerfile .

docker login

docker push  abhijdock123/requester-services-dx:uat-13-11-25-5
